做学习用
////////////////////////////////////////////////////////////////////////////

该日志系统定义了各种日志等级 (错误等级：警告、错误、信息等等)；

日志系统的输出途径基于自己设置的Appender的种类，例如Console、File等等

当需要新的输出位置的时候，需要额外的继承自Appender同时重载相应的函数；

main中给出了该日志系统最简单的使用方式。
